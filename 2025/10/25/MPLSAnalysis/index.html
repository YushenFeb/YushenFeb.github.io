基于Tshark的MPLS协议BAT分析工具 MPLS Protocol BAT Analysis Tool Based on Tshark

一、MPLS协议简介

MPLS（多协议标签交换）是一种高性能、可扩展的、独立于协议的数据转发技术，它在OSI模型的第2层（数据链路层）和第3层（网络层）之间运行，因此被称为2.5层技术。

1.MPLS的核心思想

MPLS的主要目标是结合第2层交换的简单、快速特性和第3层路由的灵活特性。

  • 传统 IP 转发： 路由器对每个数据包都进行复杂的最长匹配路由表查询，耗时较长（尤其是在早期硬件受限时）。
  • MPLS 转发： 在进入MPLS域时，通过一次路由查找将IP报文归类，并在其前面添加一个短而定长的标签。在MPLS域内，设备只需根据标签进行简单的交换（查找LFIB），无需再次检查复杂的IP报头，从而加速转发。

2.关键概念与术语

  • 标签(Label)：一个4字节的标识符，其中20比特是标签值。它只有本地意义（不同设备上标签值可以不同），用于标识一个FEC。
  • 标签栈(Label Stack)：MPLS报文可以携带多层标签，这些标签以堆栈形式存在，决定转发的始终是栈顶标签。
  • 标签头(Label Header)：4字节的MPLS封装，位于二层头和三层头之间。其中包含Label(20Bits),EXP/TC(3Bits)用于QoS,S(1Bit)栈底位,TTL(8Bits)生存时间。
  • 转发等价类(Forwarding Equivalence Class,FEC)：具有相同转发处理方式的一组报文（例如，所有发往同一目的地址的报文）。一个FEC对应一条LSP。
  • 标签交换路径（Label Switched Path,LSP）：报文在MPLS网络中的一条单向转发路径，从入口LER到出口LER。LSP需要先建立才能转发数据。
  • 标签交换路由器（Label Switch Router,LSR）：MPLS网络中的核心设备，负责基于标签进行转发（交换/Swap）。
  • 标签边缘路由器（Label Edge Router,LER）：又称 PE，位于MPLS网络的边界，连接IP网络，负责标签的压入(Push)和弹出(Pop)。
  • 标签分发协议（Label Distribution Protocol,LDP）：最常用的控制平面协议，用于协商和分发标签，建立LSP。

3.标签转发动作

数据包在MPLS网络中主要经历压入、交换和弹出三个基本操作，详细内容如下表所示。

动作 英文 发生位置 描述
压入 Push/Impose 入口LER 收到IP报文，查找FEC，在IP报文前添加（压入）第一层或多层MPLS标签。
交换 Swap 中间LSR 收到带标签报文，根据入标签查找LFIB，将入标签替换为出标签，转发给下一跳。
弹出 Pop/Dispose 出口LER或倒数第二跳LSR 移除标签，恢复成IP报文或下一层标签，转发给目的地。
倒数第二跳弹出 PHP(Penultimate Hop Popping) 出口LER的上游LSR 一种优化机制，在到达出口LER之前的一跳，就将标签弹出，减轻出口LER的处理负担。

4.MPLS优势与应用

  • 流量工程(TE)：可以强制数据流沿非最短路径转发，实现流量均衡和优化。
  • 服务质量(QoS)：利用标签头中的EXP字段（3Bits）对不同业务流进行分类和优先级处理。
  • VPN服务：MPLS L3VPN（最主要的应用）和MPLS L2VPN，为客户提供隔离和互联的虚拟私有网络服务。
  • 高性能：早期通过简化的标签交换查找，显著提高了转发效率（尽管随着硬件发展，这一优势已不那么明显，但其隧道能力仍是核心）。

二、基于Tshark的MPLS协议BAT分析工具

1.需求来源

网络数据分析工程师在使用Wireshark对网络抓包数据开展分析时，往往需要对Pcap/Cap数据中的用户进行聚类分析。最常用的方式便是通过MPLS协议对数据中的用户进行区分，再将具有同样MPLS标签组合的数据进行提取。具有相同MPLS标签组合的数据，通常被视为是同一用户所产生的数据，可对同一用户的数据进行聚类分析。在面对多用户混合数据且数据量较大时，聚类分析工作往往费时费力。在使用Tshark对Pcap/Cap数据依据MPLS标签组合进行分片操作时，也难以针对批量cap/Cap数据中多个MPLS组合进行自动化MPLS标签统计分析及数据分片聚类操作。针对此数据分析瓶颈，使用BAT语言编写了基于Tshark的MPLS协议分析工具（MPLS Analysis）。使用BAT语言进行开发的主要原因是，BAT语言无需额外的依赖环境，便于更加方便的直接移植使用。同时该基于Tshark的MPLS协议分析工具（MPLS Analysis）共具有两个版本，数据分析版本为MPLS Analysis，统计分析版本为MPLS Analysis Only Txt。
